# LOCALIZADOR DE PRÉ DEVOLUÇÃO
<b> O aplicativo foi desenvolvido em Python 3.12.7.
  <br>
<b> Alguns usuários solicitaram uma maneira mais prática de identificar qual documento está vinculado a uma determinada pré-devolução, sem a necessidade de sair da tela em que estavam trabalhando no sistema.
  <br>
<b> Como a alteração direta no sistema não era viável, a solução mais adequada foi o desenvolvimento de um aplicativo independente e leve, focado exclusivamente nessa necessidade específica.
